# AI Resume Analyzer

## 📌 Description
AI Resume Analyzer is a web application that compares resumes with a job description and ranks them based on relevance using keyword matching.

## 🚀 Features
- Upload resume (PDF/Text)
- Input job description
- Keyword extraction
- Resume scoring & ranking

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS
- Basic NLP (text processing)

## ⚙️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/AI-Resume-Analyzer.git

2. Navigate to project:
   cd AI-Resume-Analyzer

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000/
